# grdk-core

GRDKCore - Biblioteca principal dos serviços Docker

## Portas dos Serviços

* GRDKProxy (80, 8080 e 443)
* GRDKRepo (8000 e 2200)
* GRDKRepoDI (5000)
* GRDKMonitor (8001 / 9090, 9100 e 3000)
* GRDKMsg (8002)
* GRDKBackup
* GRDKN8N (5678 e 5679)

## Pré-requisitos

* Ubuntu Server 20.04
* GRDKInstallSkel
